# Master Thesis — Cahn-Hilliard Phase Field Model

Simulazione numerica dell'equazione di Cahn-Hilliard per lo studio 
della riorganizzazione dei pori nel silicio. 
Sviluppato nell'ambito della tesi magistrale in Fisica, 
Università degli Studi di Milano-Bicocca.

## Struttura della repo

```text
master_thesis/
├── cahn_hilliard_equation/   # Codice principale della tesi
│   ├── src/cahn_hilliard/    # Moduli core (operatori, condizioni iniziali, ecc.)
│   ├── scripts/              # Script per lanciare simulazioni e dataset
│   └── results/              # Output delle simulazioni
└── preliminary/              # Lavoro propedeutico
    ├── allen_cahn_equation/
    └── heat_equation/
```
